                    <li>국내/해외 뉴스에 AI 기반 **기업 태깅, ESG, 긍/부정 판단** 등 적용 가능</li>

                </ul>

                    <li>기업 중대 사건을 뉴스, 공시, 리포트와 연결하여 **리스크 및 투자 방향** 데이터 제공</li>
                    <li>내부 데이터와 외부 데이터를 통합하여 **검색 및 질의 응답** 가능한 지식 인프라 구축</li>
                    <li>**방대한 데이터를 여러 방면으로 연결**하여 유의미한 인사이트 도출 가능</li>
                </ul>
